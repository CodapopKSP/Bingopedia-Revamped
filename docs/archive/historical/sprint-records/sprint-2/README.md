## Sprint Goals

1. **G1 – Make Table of Contents functional and performant**
   - Table of Contents must load instantly and display correctly formatted content
   - ToC links must reliably navigate to article sections

2. **G2 – Fix image interaction bugs**
   - Prevent images from triggering navigation when clicked

3. **G3 – Improve game state communication clarity**
   - Timer pause indicator should be less distracting and more contextually placed

4. **G4 – Enhance leaderboard game details modal UX**
   - Bingo board view should be fully visible and properly sized within the modal

## Key Deliverables

- ✅ S1: Table of Contents performance optimization (2-3 second delay → <100ms)
- ✅ S2: Table of Contents display and functionality fix (correct section titles, navigation)
- ✅ S3: Image click prevention (images don't trigger navigation)
- ✅ S4: Timer pause indicator placement (contextual under loading message)
- ✅ S5: Leaderboard modal sizing fix (full 5×5 board visible)
