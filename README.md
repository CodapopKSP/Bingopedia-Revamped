# Bingopedia

Wikipedia Bingo game - navigate from a random starting article to find 5 articles in a row on your bingo grid.

## Quick Start

1. **Environment**: Create `.env.local` with MongoDB credentials (see `QUICK_REFERENCE.md`)
2. **Frontend**: `cd app && npm run dev`
3. **Backend** (local dev): `npm run dev:server`

See `QUICK_REFERENCE.md` for detailed setup instructions.
